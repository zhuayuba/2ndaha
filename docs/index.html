旅行前冻一份披萨，是给未来自己的礼物

出发旅行前，点你最喜欢的披萨，吃一部分，剩下的密封好放进冷冻室。一周后你深夜到家，筋疲力尽，打开冰箱只有鸡蛋、泡菜和调料——然后你想起冷冻室里那份披萨。这不是关于披萨，而是关于：你可以在某个时刻，为一个你还不认识的自己提前做一件小事。那个未来的你，会因此觉得此刻的自己值得被爱。

社交媒体是社交的反面

我们一直以为社交媒体是社交生活的低配替代品——方便但质量差。但实际它不是在模拟社交，而是在消解社交。真正的社交让你和他人建立连接；社交媒体训练你把他人变成观众，把互动变成表演。你用完社交媒体后感觉更孤独，不是因为你'没真的在社交'，而是因为它反向运行了你的社交本能。

乐观不是情绪，是纪律

115年前，Christian Larson写了最早的自我宣言之一《乐观者信条》。它说乐观不是一种心情，而是一种选择、一个有用的承诺。不是对世界或老板承诺，而是对自己承诺。乐观不是你碰巧感到的东西，是你每天训练自己做出的决定。就像健身，你不能等有动力了才去练——你得在不想练的时候也练。
