麻醉中大脑仍在学语言

📌 研究发现，人在全麻状态下，大脑海马体仍能处理复杂语言任务，包括区分词性和预测即将出现的词语。这推翻了意识与认知能力必须同步的常识，表明认知可能独立于意识运行。

贝勒医学院的科学家干了一件特别反常识的事：他们给全麻的病人脑袋里插了高精度探针，发现就算人睡得跟死猪一样，大脑还在偷偷干活。以前我们都觉得，麻醉了就是意识没了，认知功能肯定也跟着罢工。但实验啪啪打脸。 他们先给病人放一串重复的嘟嘟声，偶尔插个怪声进去。结果海马体里的神经元居然能精准识别这些异常声音，而且越听越准——这相当于大脑在麻醉状态下还在学习。更绝的是，他们后来直接放短故事，发现大脑能实时区分名词、动词和形容词，甚至能预测下一个词是什么。 这推翻了什么？我们一直以为意识是认知的开关：醒了才能思考，睡了就死机。但实验证明，语言理解、预测这些高级功能，根本不需要你醒着。大脑就像个自动运行的服务器，你关机了它还在后台跑程序。 为什么重要？如果认知可以脱离意识存在，那植物人、深度睡眠者可能还有我们不知道的思维活动。这甚至可能改写我们对意识的定义——意识可能不是认知的源头，而是认知的副产品。

生命不是机器，它有目标

📌 生物学家提出「生物能动性」理论，认为生命体与机器的本质区别在于拥有内在目标——细胞主动维持自身、生长繁殖，而非被动执行程序。这挑战了将生命视为机械过程的传统观点。

几百年来，生物学家一直把生命当机器研究：细胞是零件，DNA是程序，生物反应就是物理化学的机械过程。但最近一群科学家跳出来说：别扯了，生命和机器的根本区别在于——它有目标。 你可能会说，细菌朝食物游过去不就是化学梯度反应吗？但新理论认为，这不是被动反应，而是主动追求。细胞不是在执行程序，而是在努力活下去、长大、生孩子。胚胎发育也不是在运行基因代码，而是在朝着一个完整形态努力。 这听起来有点玄学？其实非常科学。传统生物学之所以回避「目的论」，是因为怕沾上宗教色彩。但复杂系统理论和进化生物学证明，不承认生物有内在目标，你就解释不了为什么癌细胞会「选择」疯狂增殖，为什么胚胎会「坚持」长成特定形状。 这个转变有实际价值：研究癌症时，不再把癌细胞当失控机器，而是看作有「错误目标」的活物；造AI时，要思考机器能否真的拥有目标；甚至重新定义什么才算生命。简单说：生命不是被动的零件组合，而是自带欲望的主动玩家。
