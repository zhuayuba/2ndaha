你有没有遇到过那种人——或者你自己就是——觉得'不拼命就不算努力'？比如减肥必须饿到头晕才算有效，工作必须加班到深夜才算认真，学习必须熬夜背书才算刻苦。原文管这叫'意志力神话'：一种认为'美德在于用极端努力克服障碍'的信念。 但作者说，这种信念其实是'低能动性'的表现。什么意思？就是当你没有能力或不愿意改变环境、优化方法时，你只能靠死磕自己来获得成就感。比如你明明可以找个更高效的工作流程，但你偏要熬夜加班，因为这样让你感觉自己'很努力'。作者举了个例子：关于Ozempic（一种减肥药）的争议——很多人觉得用药物减肥是'作弊'，必须靠节食和运动才算'真正的减肥'。但作者反问：如果药物能安全有效地帮你达到目标，为什么非要选择痛苦的那条路？ 这个视角推翻了我们从小被灌输的'天道酬勤'。它不是说努力不重要，而是说：崇拜努力本身，往往是因为你找不到更好的方法。真正的高手追求的是'聪明地省力'，而不是'拼命地受苦'。下次你觉得自己'不够努力'时，也许该问的是：我是不是在用努力来逃避思考？
