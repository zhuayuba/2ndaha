社交媒体是社交的反面

我们一直以为社交媒体是社交生活的低配替代品——方便但质量差。但实际它不是在模拟社交，而是在消解社交。真正的社交让你和他人建立连接；社交媒体训练你把他人变成观众，把互动变成表演。你用完社交媒体后感觉更孤独，不是因为你'没真的在社交'，而是因为它反向运行了你的社交本能。
